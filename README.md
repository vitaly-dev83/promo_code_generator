# Промо-генератор и валидатор

Веб-утилита для генерации промокодов с контрольной суммой и проверки их валидности.

## Особенности
- Генерация кодов заданной длины (4–20 символов)
- Опциональное включение цифр
- Контрольная сумма (последний символ вычисляется из предыдущих)
- Копирование в буфер обмена
- Валидация с проверкой допустимых символов

## Технологии
HTML5, CSS3, JavaScript (Vanilla)

## Запуск
Откройте index.html в любом браузере
