## 🧠 Как работает контрольная сумма

Контрольная сумма — это специальный символ в конце кода, который вычисляется из предыдущих символов.

**Как это работает:**
1. Каждый символ кода преобразуется в числовой код
2. Все коды складываются
3. Берётся остаток от деления на количество допустимых символов
4. По этому остатку выбирается контрольный символ

**Зачем это нужно:** Если изменить любой символ в коде, контрольная сумма перестанет совпадать — и валидатор покажет ошибку. Это защита от случайных или намеренных изменений.
